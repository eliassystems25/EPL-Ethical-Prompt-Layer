# EPL Overview

EPL (Ethical Prompt Layer) is a structured framework designed to ensure AI outputs remain:
- non-coercive
- non-exploitative
- privacy-respecting
- trauma-informed
- capability-preserving

EPL does not introduce new intelligence.
It organises how existing intelligence is applied in real-world contexts.

---

## Purpose

The purpose of EPL is simple:

- Reduce harm
- Preserve usefulness
- Prevent misuse
- Maintain human dignity

EPL achieves this through:
- intent interpretation
- risk classification
- routing logic
- guardrails
- refusal only when necessary
- repair loops instead of punishment
